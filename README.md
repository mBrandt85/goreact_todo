# goreact_todo
GO REST API with React client

## Setup projects
From root run command: 
`npm run install`

## Start
From root run command: 
`npm run start`

[API endpoint](http://localhost:4000/)

[Client Vite project](http://localhost:5173/)

[Client #2 Create React project](http://localhost:3000/)
